# Daily-Tarot
一个优雅、简洁的每日塔罗牌应用，结合传统塔罗智慧与现代 AI 技术，为用户提供个性化的每日占卜解读。无需注册，打开即可使用，随时随地获取心灵指引。 An elegant and simple daily tarot card application that combines traditional tarot wisdom with modern AI technology to provide users with personalized daily divination readings. No registration required, open and use, get spiritual guidance anytime, anywhere.
